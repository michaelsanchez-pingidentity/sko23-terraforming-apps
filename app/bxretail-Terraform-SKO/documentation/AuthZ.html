Class: AuthZ

AuthZ()

Class representing OAuth authorization operations. This demo-specific class is developed and maintained by Ping Identity Technical Enablement. Implements methods to process UI requests and data to send to Ping integration methods. This is a modified version of what comes with the live BXRetail app. This component was stripped down to only what was necessary for the Terraform labs.
